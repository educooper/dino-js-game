## :coffee: Addicional Features / Recursos Adicionais 

Looping de animação do caminhar do personagem

```css
.dino {
  position: absolute;
  bottom:0;
  background-image: url(dino-sprite.png);
  width: 56px;
  height: 60px;
  animation: play 0.2s steps(4) infinite;
  }
  @keyframes play {
    100% {
      background-position: -224px;
    }
  }

```
